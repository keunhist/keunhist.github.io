찾아오시는 길

세종청사 오시는길

주소

  • 주소 : 세종특별자치시 갈매로 388 정부세종청사 15동
  • 대표전화 : (044) 203-2000 (월~금 09:00~18:00, 공휴일 제외)
  • 팩스 : (044) 203-3447

교통편

승용차

내비게이션에 "정부세종청사" 또는 "문화체육관광부"로 검색

  • 경부고속도로
    • 남청주 IC → 부강 → 국지도 96호 → 정부세종청사
    • 청주 IC → 36번 국도(조치원 방향) → 정부세종청사
  • 호남고속도로
    • 유성 IC → 1번 국도(조치원 방향) → 정부세종청사
    • 남세종 IC → 1번 국도 (조치원 방향) → 대평리 → 1.0km 가량 직진후 우회전
  • 논산천안고속도로
    • 정안 IC → 정부세종청사 연결도로 → 은하수공원 → 정부세종청사
  • 당진영덕고속도로
    • 서세종 IC(동공주 IC) → 36번 국도 → 정부세종청사
    • 남세종 IC(북유성 IC) → 1번 국도(조치원 방향) → 정부세종청사

기차

  • 대전역 출발
    • B1(BRT)(대전역) → 정부세종청사남측 정류장 하차
    • 지하철 승차(반석행) → 급행버스 B2(BRT.오송역방면)(반석역) → 정부세종청사남측 정류장 하차
    • 지하철 승차(반석행) → 급행버스 1000(홍익대학교방면)(반석역) → 어진중학교 정류장 하차
  • 조치원역 출발
    • 601(조치원.대평)(조치원역) → 정부세종청사남측 정류장 하차
    • 1000(반석역방면)(조치원역뒤편) → 어진중학교 정류장 하차
    • 991(세종국책연구단지방면)(조치원역뒤편) → 정부세종청사남측 정류장 하차
    • 500(산성동 방면)(조치원전통시장) → 정부세종청사남측 정류장 하차
    • 550(산성동 방면)(조치원역) → 정부세종청사남측 정류장 하차
  • 오송역 출발
    • B2(BRT.반석역방면) → 정부세종청사남측 정류장 하차
    • B1(BRT.대전역방면) → 정부세종청사남측 정류장 하차
    • 751번(세종터미널방면) → 정부세종청사남측 정류장에서 하차

시외버스

  • 세종고속시외버스터미널 (070-4904-3263)
    • B2(BRT.오송역방면) → 정부세종청사남측 정류장 하차
    • B1(BRT.오송역방면) → 정부세종청사남측 정류장 하차
